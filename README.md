# Employee Payroll System

A C# console application that demonstrates object-oriented programming by managing employee payroll information with password-protected access to salary details.

## Features
- Employee and Full-Time employee classes
- Weekly wage calculation from yearly salary
- Password-protected salary information
- Password change functionality
- Inheritance and method overriding
- Console-based user interaction

## Technologies
- C#
- .NET Console Application

## How It Works
The program creates full-time employee objects, calculates weekly wages from yearly salaries, and allows users to securely view salary information after password authentication. Employees can also update their passwords through the console interface.
